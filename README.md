## This is a Vue.js project

### Project setup

Clone the repo:

```
git clone git@github.com:keithmchd48/scrimba-film-db
```
Install dependencies:

```
Run npm install
```

Run the project on localhost:

```
npm run serve
```

#### API used for fetching movies

[The Movie DB](https://api.themoviedb.org)
